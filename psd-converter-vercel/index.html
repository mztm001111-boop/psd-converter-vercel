浏览器兼容性提示

"保存到指定文件夹"功能基于 File System Access API，目前在 Chrome 86+ / Edge 86+ 等现代浏览器中可用。Firefox、Safari 请使用"ZIP 下载"或"逐个下载"方式。

大文件贴士（偶发场景）

  • 推荐单个文件 ≤ 800MB（PSD 尤其受限）；超过 300MB 会自动降采样（最大边长压到 8192px，超 800MB 压到 4096px）。
  • 处理大文件时推荐使用 "保存到指定文件夹" 模式——转换后立即写盘并释放内存，可连续处理多个大文件。
  • 超过 2GB 的单文件受浏览器 ArrayBuffer 上限限制，无法处理，请使用桌面工具预处理。
  • TIFF 多页文件目前只转换第一页；ICO 多尺寸图标将自动选择最大尺寸帧。
  • 如果转换中途卡顿，建议关闭其他占内存的标签页，或将"最大边长"手动调到 2560px 以下。
